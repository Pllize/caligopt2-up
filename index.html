• 📱 폰을 교체하거나 기기를 바꿀 때
• 🗑 브라우저 캐시·사이트 데이터를 삭제할 때

💡 기기 이전 방법:
① 지금 기기에서 백업하기 → JSON 파일 저장
② 새 기기에서 이 페이지 접속
③ 불러오기로 저장한 JSON 파일 선택

⚠️ 데이터는 브라우저 로컬에 저장돼요.

백업 파일로 불러오면 현재 기기의 모든 데이터가 덮어씌워져요.
